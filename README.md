# Data-Driven Estimation of Residential EV Charging Loads: Houston, TX

## Features
The data repository offers a realistic residential electric vehicle (EV) charging load data for Houston, TX area. The key features include:

### Sampled EV charging load profile of passenger car in Texas
- **Resolution**: 10-minute interval resolution
- **Data Structure**: `start_datetime`, `stop_datetime`, `duration`, `energy_kwh`, `soc_start`, `soc_stop`
- **Data source**: EV Watts
